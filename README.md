# Alchemist

This repository contains the Next.js frontend dashboard for the Alchemist DevSecOps platform as requested.

## To Run the Dashboard

1. Navigate to the web app directory:
   ```bash
   cd apps/web
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

The Overview page (`/`) and Projects page (`/projects`) have been built to match your design specifications!
